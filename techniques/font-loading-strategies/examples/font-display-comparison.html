Value Block Period Swap Period Behavior Best For
auto ~100ms ~3s Browser default behavior -
block 2-3s ∞ Text invisible until font loads Brand-critical fonts
swap ✅ 0ms ∞ Show fallback immediately, swap when ready Most sites (recommended)
fallback ~100ms ~3s Brief block, then swap with timeout Balance performance/brand
optional ~100ms 0ms Use font only if already cached Performance-critical sites
