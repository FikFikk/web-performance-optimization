Pros & Cons

Advantages Disadvantages
✅ Zero network overhead ❌ Inconsistent cross-platform
✅ Instant rendering ❌ Limited branding control
✅ No FOIT/FOUT ❌ No custom styles
✅ Zero CLS ❌ Design differences
✅ Familiar to users
✅ Excellent legibility
✅ Performance perfection

When to Use System Fonts

  • Content-heavy sites — Blogs, news, documentation
  • Performance-critical apps — E-commerce, dashboards
  • Internal tools — Admin panels, corporate apps
  • MVP/Prototypes — Ship fast, optimize later
  • Accessibility-first — Government, education sites

Who Uses System Fonts?

  • GitHub — Entire UI menggunakan system fonts
  • WordPress admin — Performance & accessibility
  • Medium (partial) — Body text uses system fonts
  • Stack Overflow — Fast, functional, familiar
  • Apple.com — San Francisco on Apple devices

Recommendation: Start dengan system fonts. Add custom fonts hanya jika brand identity benar-benar membutuhkannya. Performance > perfection.
