Halaman Demo - Setelah Optimasi Script Pihak Ketiga

[Gambar LCP / Hero Image Halaman]

Halaman ini sekarang mengimplementasikan teknik optimasi Third-Party Script Loading. Script analitik sekarang dimuat dengan tag async sehingga browser tidak terhenti memparsing HTML.

Yang paling krusial, Chat Widget Pelanggan yang berat tidak lagi digarap di pemuatan halaman awal. Sebagai gantinya, kami menampilkan tombol placeholder Chat terapung (berukuran kurang dari 1KB HTML/CSS). Script utama widget chat (dimensi 200KB+ & CPU block 350ms) baru akan dimuat secara dinamis saat pengguna mengarahkan kursor/mengklik tombol chat, atau ketika browser dalam keadaan menganggur (idle).

Teknik Optimasi yang Diimplementasikan:

  • Resource Hints: preconnect ke host script pihak ketiga.
  • Asynchronous loading: analytics loading non-blocking menggunakan async.
  • Lazy Loading & Delay Execution Chat: Script chat dimuat saat interaksi user (hover/click) atau event idle (requestIdleCallback).
